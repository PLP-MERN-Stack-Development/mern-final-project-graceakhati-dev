# Planet Path Hero Section - Image Specifications

## Design Style
- **Style**: Soft illustrated, playful, friendly (not realistic)
- **Color Palette**: Soft pastel colors
- **Composition**: Mobile-friendly, landscape orientation
- **Mood**: Warm, inviting, optimistic

## Color Palette
- Primary Green: #2E7D32 (dark green)
- Light Green: #A5D6A7 (soft green)
- Earth Brown: #8D6E63 (warm brown)
- Light Beige: #D7CCC8 (soft beige)
- Sky Blue: #B3E5FC (soft blue)
- Sun Yellow: #FFF9C4 (soft yellow)
- Cloud White: #FFFFFF

## Technical Requirements
- **Format**: PNG for images (with transparency where needed), SVG for icons
- **Optimization**: Compressed for web (under 500KB per PNG, under 50KB per SVG)
- **Transparency**: PNG files should support alpha channel where needed
- **Responsive**: Images should look good at various sizes (mobile to desktop)

## Usage Notes
- All images should be saved in `/client/src/assets/images/`
- Icons should be saved in `/client/src/assets/icons/`
- Use descriptive alt text when implementing
- Consider lazy loading for hero images
